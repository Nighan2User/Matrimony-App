# Matrimony App (Flutter)

This repository contains the Flutter source code for the Matrimony application.

## Project Structure

- matrimony-flutter-app/
  - lib/        → Flutter UI & logic
  - android/    → Android build files
  - ios/        → iOS build files
  - web/        → Flutter web build
  - macos/ / windows / linux → Desktop support

## Tech Stack

- Flutter (Web, Android, iOS)
- Dart
- Firebase

## Notes

This repo contains source code only. Generated files and dependencies are ignored via `.gitignore`.
